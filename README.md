 🏀 Basketball Scoreboard

A simple basketball scoreboard built with **HTML, CSS, and JavaScript**.

 Features

* Add points (+1, +2, +3) for each team
* Track fouls
* Game timer (start, pause, reset)
* Period counter
* Leader highlight
* New game reset

 Tech Stack

* HTML
* CSS
* JavaScript

 ▶️ How to Run

Open `index.html` in your browser.

## 💡 About

This project was built to practice JavaScript DOM manipulation and event handling.
